# ADAS_yolo

### 1. Description ###
ADAS구현을 위해 객체 인식 연습하기
1. 블랙박스 영상 가져와서 YOLO v4 모델로 객체 인식하기 -> 'result.avi'파일로 결과 영상 저장
2. Webcam으로 실시간 영상 가져와서 객체 인식하기
